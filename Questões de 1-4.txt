1 – Com suas palavras descreva o que é a DOM e qual sua relação com o HTML?

R: O Document Object Model (DOM) é uma interface de programação para os documentos HTML e XML. Representa a página de forma que os programas possam alterar a estrutura do documento, alterar o estilo e conteúdo.

2 - O que é o canvas HTML?

R: O elemento HTML Canvas ( <canvas> ) pode ser utilizado para desenhar gráficos utilizando scripts (geralmente JavaScript).

3- – Quais as linguagens e tecnologias comumente envolvidas no desenvolvimento de produtos baseados em
canvas html? 

R: A linguagem mais usada por esse recurso é a linguagem Javascript, que permite a criação de animações através de códigos de programação dinâmica, além do canvas também ser utilizado para desenhar gráficos e ser utilizado para fazer composições de fotos e também para animar formas e imagens.

4- Explique para que serve o código “document.getElementById()”

R:  É uma função que serve para retornar um elemento do DOM que é identificado por um ID específico.
